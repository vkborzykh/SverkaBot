# SverkaBot — сверка выплат Wildberries с банковскими выписками

**SverkaBot** — Telegram-бот для автоматической сверки выплат Wildberries с банковскими выписками. Помогает продавцам на маркетплейсах быстро находить недостающие выплаты, выявлять несовпадения и оценивать возможные потери.

## 📌 Основные возможности

- 📂 Загрузка отчёта Wildberries (XLSX)
- 🏦 Загрузка банковской выписки (XLSX, CSV) – автоматическое распознавание структуры
- 🔄 Автоматическая сверка по точной сумме, дате и другим параметрам
- 📊 Детальный отчёт в формате ZIP с 9 CSV-файлами (совпадения, расхождения, метрики)
- 🧠 Умные профили банков – система обучается на новых шаблонах выписок
- 💰 Расчёт потенциальных потерь на основе несовпадений
- 📜 История сверок – доступ к предыдущим отчётам
- 🔐 Пробный период 7 дней, затем платная подписка (1500₽/30 дней)
- 🛡️ Полное удаление данных по запросу пользователя

## 🚀 Технологический стек

| Компонент | Технология |
|-----------|------------|
| Фронт/Бэк | Next.js 14 (App Router), TypeScript |
| База данных | PostgreSQL (Supabase) |
| ORM | Drizzle ORM |
| Telegram API | Telegraf |
| Парсинг Excel/CSV | xlsx, papaparse, iconv-lite |
| Отчёты | archiver (ZIP), CSV с русскими заголовками |
| Хостинг | Vercel |
| Аутентификация | Telegram webhook secret + internal tokens |
